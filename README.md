# MindGarden

> Learn deeply. Think clearly. Grow consistently.

## Overview

MindGarden is an AI-powered learning operating system designed to help students, developers, and lifelong learners build knowledge intentionally rather than simply collect information.

Unlike traditional note-taking or productivity applications, MindGarden treats knowledge as a living system. Every concept evolves over time based on understanding, revision, and application. Instead of rewarding users for creating more notes, MindGarden focuses on helping them retain knowledge, identify learning gaps, and develop long-term mastery.

This project is being developed incrementally as part of my software engineering journey. Every programming concept I learn is immediately applied to improve MindGarden, allowing the project to evolve from a simple Java console application into a modern full-stack AI-powered learning platform.

---

## Problem Statement

Most productivity applications focus on organizing information through notes, tasks, and reminders. While they are excellent for storing information, they rarely help users understand, retain, or revisit knowledge effectively.

Students and professionals often experience problems such as:

- Creating notes that are never reviewed.
- Forgetting previously learned concepts.
- Scattered learning resources across multiple platforms.
- Lack of meaningful feedback about learning progress.
- Passive note-taking without active recall.

MindGarden aims to solve these problems by making learning intentional, measurable, and continuous.

---

## Vision

The vision of MindGarden is to become a learning operating system that enables users to capture knowledge, build understanding, strengthen memory, and develop consistent learning habits.

The platform combines knowledge management, productivity, artificial intelligence, and learning science into a single workspace designed for continuous personal growth.

---

## Core Concept

MindGarden models knowledge as a living ecosystem rather than a static collection of notes.

### Knowledge Growth

Every note progresses through different stages of understanding.

```
Seed → Sprout → Tree
```

Growth is determined by learning activity, revision, and demonstrated understanding rather than manual categorization.

### Knowledge Decay

Concepts naturally weaken over time if they are not revisited.

Instead of hiding forgotten topics, MindGarden surfaces them using a decay model inspired by the forgetting curve, helping users focus on what actually needs attention.

### Knowledge Graph

Every concept can connect to related concepts through a relationship graph.

Rather than maintaining isolated notes, users gradually build an interconnected knowledge network that reflects how understanding develops over time.

### Learning Rituals

Each focused study session begins with a learning intention and ends with a reflection.

This encourages deliberate practice instead of passive consumption.
